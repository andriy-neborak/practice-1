# UART Echo Server (6-byte Packet)

Проста прошивка для **STM32F051R8**, що реалізує стабільний UART ехо-сервер з буферизацією пакетів фіксованої довжини.

## ⚙️ Логіка роботи

Цей код реалізує механізм "Store and Forward" для пакетів даних:

1.  **Прийом (Non-blocking):**
    * Дані приймаються по 1 байту через переривання (`HAL_UART_Receive_IT`).
    * Основний цикл процесора (`while(1)`) залишається вільним для інших задач.

2.  **Буферизація:**
    * Отримані байти складаються у масив `echo_buf`.
    * Лічильник `rx_idx` відстежує кількість прийнятих байт.

3.  **Ехо-відповідь (Echo):**
    * Як тільки накопичується **рівно 6 байт**, контролер відправляє весь пакет назад на ПК (`HAL_UART_Transmit`).
    * Індекс буфера скидається, починається прийом нового пакету.

4.  **Захист (Error Handling):**
    * Реалізовано колбек `HAL_UART_ErrorCallback` для обробки помилок переповнення (Overrun Error).
    * Це запобігає "зависанню" інтерфейсу при швидкій передачі даних.

## 🛠 Технічні деталі
* **MCU:** STM32F051R8
* **Interface:** UART1
* **Baud Rate:** 38400 (стандартно)
* **Packet Size:** 6 Bytes

---
*Цей модуль використовується для перевірки цілісності зв'язку між ПК та Мікроконтролером.*
